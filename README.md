# 星空熊抱大冒險

一個繁體中文 3D 星空跳躍閱讀問答遊戲，題目圍繞《熊抱青春記》的角色、情節及成長主題。

## 遊戲內容

- 開始前輸入學生姓名
- 共 10 題繁體中文選擇題
- 跳上正確答案星星，逐層前往故事書城堡
- 答錯或跌落會扣除 1 條生命，共有 5 條生命
- 鍵盤方向鍵控制左右移動
- 空白鍵跳躍
- 使用滑鼠點擊發光禮物，可隨機得到 60 秒翅膀、巨人或小貓能力
- 結束後提交姓名、分數、答對題數、剩餘生命及完成時間至 `yuetki1999@gmail.com`

## 開啟遊戲

直接開啟 `index.html`，或使用 GitHub Pages：

1. 進入 Repository 的 **Settings**
2. 選擇 **Pages**
3. 在 **Build and deployment** 選擇 **Deploy from a branch**
4. Branch 選擇預設分支及 `/root`
5. 按 **Save**

## 電郵提交的重要設定

本遊戲使用 FormSubmit 傳送成績。第一次有學生提交時，`yuetki1999@gmail.com` 會收到一封啟用郵件；老師必須按郵件內的確認連結，之後成績才可自動送達。

如果自動提交失敗，結果畫面會顯示「使用電郵程式傳送成績」後備按鈕。

